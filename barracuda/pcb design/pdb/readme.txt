QUOTATION INFORMATION
---------------------
* Circuit Size: 40x30mm
* Copper weight: 2oz on internal planes only
* Material: 1.6mm FR4
* Number of holes: ~120
* Smallest hole: 0.3mm
* Largest hole: 3mm
* Min track width space: 0.15mm (in a few cases only, typically ~0.3mm)
* Solder resist - blue if possible, otherwise whatever is easiest (green)
* Silkscreen - top & bottom - white
* Surface Finish - Gold (preferably, otherwise whatever is easiest)

BOARD INFORMATION
-----------------
* 4-layer
* Dimensions: 96x115mm
* Outline defined on top silkscreen layer (please crop the connector silkscreen which extends past the board outline)
* Solder resist - blue if possible
* Silkscreen - top & bottom - white

LAYER STACK
-----------
pdb.gto - top silkscreen
pdb.gts - top solder mask
pdb.gtl - top copper layer
pdb.gp1 - internal plane 1
pdb.gp2 - internal plane 2
pdb.gbl - bottom copper layer
pdb.gbs - bottom solder mask
pdb.gbo - bottom silkscreen
pdb.txt - NC drill file
